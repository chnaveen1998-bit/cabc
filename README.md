# CABC Church — Static Template

This is a minimal static website template with a hero section (church image) and a polymorph timetable box. It includes navigation and the requested sections:

- Song book
- Sermons
- Events gallery
- Monthly Calendar
- Prayer request
- Memberships
- Fellowships
- Outreach programs
- Donations (Regular and Recurring)
- Contact us

How to open:

Open `index.html` in a browser. On macOS you can double-click the file or run:

```bash
open index.html
```

You can also serve it with a tiny static server (python):

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

Files added:

- `index.html` — main template
- `css/style.css` — styles
- `js/main.js` — small interaction helpers
